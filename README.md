# Smart Travel Planner

Java-based Smart Travel Planner application developed using Java and OpenStreetMap APIs.

## Features
- Travel itinerary planning
- Hotel recommendation system
- Budget estimation
- PDF report generation
- GUI-based interaction

## Technologies Used
- Java
- MySQL
- Eclipse IDE
- OpenStreetMap API

## Modules
- Travel Service
- Hotel Service
- Budget Service
- PDF Service
- Review System
